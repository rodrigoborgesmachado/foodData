# FoodData

FoodData é um site que permite consultar informações nutricionais de alimentos com base na tabela TACO. A plataforma foi desenvolvida para fornecer dados precisos sobre calorias, nutrientes e outros aspectos nutricionais dos alimentos disponíveis no Brasil.

## Visão Geral

- **Consulta Nutricional:** Acesse informações detalhadas sobre os alimentos, incluindo calorias, macronutrientes e micronutrientes.
- **Base de Dados TACO:** Utiliza a tabela TACO para fornecer dados precisos e confiáveis.
- **Interface Simples:** Navegação fácil para encontrar informações rapidamente.

## Tecnologias Utilizadas

- **Frontend:** React
- **Backend:** .Net 8
- **Hospedagem:** Netlify

## Funcionalidades

- **Pesquisa de Alimentos:** Pesquise alimentos por nome para obter detalhes nutricionais.
- **Dados Atualizados:** Baseado na tabela TACO, garantindo dados relevantes e precisos.
- **Visualização Intuitiva:** Interface clara e fácil de usar para visualizar informações nutricionais.
